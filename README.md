### 🚀 How to Use

1. Click the **Download** button above  
2. Extract the archive  
3. Run `ER-Player-Editor.exe`  
4. Launch **Elden Ring in offline mode**  
5. Press **INSERT** to open the menu  

**Hotkeys:**  
`INSERT` — Show/Hide menu
